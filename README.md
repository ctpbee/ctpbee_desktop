# SHT

> 打造个人自用的最好交易桌面端

#### 快速使用

```
pip install -r requirement.txt
python sht.py
```

## 文档使用

关于[ctpbee](https://github.com/ctpbee/ctpbee)

## 一些可能存在的问题

** 如果出现不存在dll载入失败 **

```
pip uninstall pyside2 
pip install pyside2
```
